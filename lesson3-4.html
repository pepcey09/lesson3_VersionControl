Pros of Git

Allows users to:

  • have multiple workers to code at the same time.
  • revert back to previous saves to edit.
  • know what users did what.
  • upload, or "push", multiple files at once.
  • merge multiple files at once.
  • retrieve, or "pull", multiple files at once.
  • work quicker due to the speedier setup of GitHub.
  • be confident in no loss of work due to the distributed VCS.
  • have an easier time sharing work with the community.
