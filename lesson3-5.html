Basic Commands of Git

This is a list including some basic Git commands and their functions.

  • git commit- staged files are sent to the repository.
  • git branch- will create a new branch.
  • git checkout- will change to the repository named.
  • git checkout -b- this will create a new branch named what is input and immediatly switch to it.
  • git reset- undos what is set to be commit to the last version.
  • git revert- used for work already commited, this will make a clone of the commit but as a new repository to not damage anything.
  • git merge- will combine current commit with target commit.
  • git rebase- will move past commits and current to the work of another seperate commit.
